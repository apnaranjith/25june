INFO ABOUT Translate property

The translate property defines x- and y-coordinates of an element in 2D. You can also define the z-coordinate to change position in 3D. .Translate property transforms the elements along x-axis,Y-axis,Z-axis i.e.;
it can change the position of the element in 2D or 3D depending on the axis,we use as the property like translateX(),translateY(),translateZ().
TranslateX():we need to mention the value in between () symbol so that it transforms the element along the X-axis.
Example:TranslateX(50px)

similiarly translateY(),translateZ().
